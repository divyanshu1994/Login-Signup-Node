-express <br>
-ejs : template <br>
-ejs-mate  <br>
-mongoose :connect to mlab <br>

-express-session <br>
-cookie-parser <br>
-body-parser : parsing body -req.body  <br>
-passport : user authentication <br>
-connect-mongo : to store session data in mongodb <br>
